# Google Play Reviewer Notes Draft (en-US)

Use for Play Console **App access** / review notes.

## App access
- **Special account needed?** No username/password account system.
- **How users join:** retreat code + display name + phone number.

## Reviewer test flow
1. Launch Calvary Caravan.
2. Enter active review retreat code (set in production before submission).
3. Enter name + phone number and join.
4. Open Map, Chat, and Profile tabs.
5. In Profile, test:
   - **Unshare my location** toggle
   - **Delete account & data** action

## Permission behavior notes
- Location is used for live convoy map updates **while the app is active**.
- Users can disable location sharing in-app at any time.
- Push notifications are optional and used for convoy updates/alerts.
